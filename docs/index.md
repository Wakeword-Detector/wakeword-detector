# Pollen

## Wakeword Detector
Free, open-source wake-word detection package using PyTorch, with associated npm package for React integrations.
STILL IN DEVELOPMENT

## Install Wakeword Detection Trainer
Train your own wakeword. Currently the package is using my model for "Willobee" detection, this package is still in development and test samples will be removed in future versions. You can delete the current samples in wakeword and not wakeword folders to use your own data and train your own wakeword. 

pip install wakeword-detector

### Usage
wakeword-record
wakeword-detect

## Integrate with React
npm install wakeword-react
